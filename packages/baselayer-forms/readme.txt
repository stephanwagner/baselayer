Form builder, submissions, notifications, and spam protection for WordPress.

== Description ==

BaseLayer Forms lets you build forms in the WordPress admin, place them with a block, store entries, send notification emails, and protect submissions with honeypot, timing, rate limits, and optional CAPTCHA.

== Installation ==

1. Upload the `baselayer-forms` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins screen
3. Open Forms in the admin menu to create a form
4. Insert the Form block on a page
